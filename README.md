# SharePoint AI

`SharePoint AI` ist meine Sammlung aus Beispielen, Skills, Lösungen und praktischen Erfahrungen rund um die Nutzung von KI in Microsoft SharePoint.

## Purpose

This repository collects examples, skills, solutions, and practical experience around using AI in Microsoft SharePoint.

## Inhalte

- künftige `docs/`
  Einordnungen, Learnings, Governance-Muster und Praxisnotizen
- künftige `samples/`
  kleine Beispiele, Strukturen und wiederverwendbare Bausteine

## Für wen dieses Repository gedacht ist

- SharePoint- und Microsoft-365-Verantwortliche
- Teams für Wissensmanagement, Kommunikation und Dokumentation
- Fachbereiche mit Bedarf an reproduzierbarer KI-Unterstützung
- Projektteams, die einen pragmatischen Einstieg statt einer reinen Tool-Demo suchen

## Sprache

Das Repository kann gemischte Inhalte in Deutsch und Englisch enthalten. Wo sinnvoll, werden zentrale Dokumente in beiden Sprachfassungen bereitgestellt.
